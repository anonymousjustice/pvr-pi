Description:
Generates a hash from generic data. This function uses the 32-bit Fowler/Noll/Vo algorithm which trades efficiency for slightly increased risk of collisions. This algorithm is public domain. More information can be found at: http://www.isthe.com/chongo/tech/comp/fnv/.

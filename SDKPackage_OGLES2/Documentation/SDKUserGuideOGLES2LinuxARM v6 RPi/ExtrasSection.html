Extras

PODPlayer

The PODPlayer is an application that displays POD files as produced and configured by our Utilities: Collada2POD, PVRGeoPOD or PVRShaman. There is an in-built menu allowing various options to be tweaked during playback and a simple script may be used to specify playback options.

Source code is provided for the PODPlayer showing how it is built upon the PVREngine classes and how this allows the same code to be built for various APIs and platforms using the appropriate PVREngine library.

More documentation is available at Extras/PODPlayer/Documentation.

The project folder for PODPlayer is available at Extras/PODPlayer.

PVREngine

The PVREngine is a collection of classes provided by Imagination to aid in the development of 3D rendering applications that builds into its own library. It is based upon the PVRShell and the PVRTools to allow further abstraction of API and OS; tries to handle common rendering functions and provides a framework upon which to build 3D applications.

The PVREngine application source, document and the comments in the engine code are provided to help in understanding the workings of the PVREngine.

More documentation is available at Extras/PVREngine/Documentation.

The project folder for PVREngine is available at Extras/PVREngine.

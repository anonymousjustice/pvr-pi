PVRTools

The PVRTools library consists of various modules designed to help developers achieve common tasks associated with hardware accelerated graphics rendering, e.g. model loading, extensions handling, matrix functions, 3D font, etc. All OpenGL ES 2.0 SDK demos make use of the OpenGL ES 2.0 Tools library.

The PVRTools source can be found in the Tools folder in the SDK package.

More documentation is available at Tools/Documentation.

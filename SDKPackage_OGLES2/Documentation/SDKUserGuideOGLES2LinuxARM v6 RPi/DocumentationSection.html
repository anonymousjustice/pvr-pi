PowerVR Series 5 SGX Architecture Guide for Developers

The purpose of this document is to provide graphics programmers with an overview of the PowerVR Series 5 (SGX & SGX-MP) graphics hardware architecture, while also highlighting why some performance recommendations make such a significant difference to graphics rendering on PowerVR platforms. Furthermore, the information in this document outlines the purpose of each hardware module for which PVRTune provides performance counters.

Parallax Bumpmapping Whitepaper

Parallax bumpmapping is a technique that adds greater detail to models without adding further geometry. This whitepaper describes the technique, the implementation available in our Training Courses, and the optimization steps taken by PowerVR Developer Technology to ensure its efficiency with PowerVR hardware.

PowerVR 3D Navigation Rendering Techniques

Describes techniques for preparing and rendering real-world navigation data efficiently in 3D. Companion to Navigation3D available in the DemosSection.html. For "2.5D" techniques please see the "PowerVR Navigation Rendering Techniques" document

PowerVR Edge Detection Training Course

PowerVR Navigation Rendering Techniques

Describes preparing and rendering real-world navigation data for rendering using PowerVR hardware acceleration. A "2.5D" approach is explored (for 3D rendering please see the "PowerVR 3D Navigation Rendering Techniques" document). This is illustrated in Navigation in our DemosSection.html.

PowerVR Post Processing Effects Development Recommendations

This document describes a general approach to post processing and provides an example of a real-time, bloom effect that can be implemented for mobile platforms utilizing PowerVR SGX hardware. The example project is one of our Training Courses.

PowerVR Shader Based Water Effects

This white paper deals with the implementation of realistic water effects on PowerVR hardware and is a companion to the Water demo to be found with our other DemosSection.html.

UI Example Whitepaper

The UI Example in DemosSection.html shows some ways to implement user interfaces efficiently using PowerVR hardware acceleration. This white paper discusses these techniques and other optimizations possible for user interfaces.

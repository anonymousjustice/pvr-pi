Installation

Prerequisites

This package assumes you are running a distribution containing Raspberry Pi source code located at /opt/vc/src. Raspbian 'wheezy' is an example of a distribution which contains these files. Other distributions are unsupported (but may work with a little modification of makefiles).

In order to compile an example you will need a C++ compiler (GCC for example) and make.

Building the Code Examples

  1. To build individual components go to the directory Demos/DemoName/OGLES2/Build/LinuxGeneric or TrainingCourse/TCName/OGLES2/Build/LinuxGeneric and run the command:

    make BUILDPI=1 PLATFORM=LinuxARMV6

    Alternatively set the following environment variables:

    export BUILDPI=1

    export PLATFORM=LinuxARMV6

    ...and simply execute:

    make

    The executables for the demos and training courses will go to: Demos/DemoName/OGLES2/Build/LinuxARMV6/ReleasePi or TrainingCourse/DemoName/OGLES2/Build/LinuxARMV6/ReleasePi where DemoName is the name of the example that you are building.

  2. OPTIONAL: It is possible to cross-compile for the Raspberry Pi by setting the TOOLCHAIN variable, and optionally CROSS_COMPILE.

    3. Running Executables

  3. Executables should be able to be ran from both the command line and under X11.
